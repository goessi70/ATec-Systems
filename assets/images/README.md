# ATEC Systems Assets

Hier können Sie alle Ihre Bilder für die Webseite speichern.

### Verwendung:
1. Speichern Sie Ihre Bilder in diesem Ordner (z.B. `hero-alarm.jpg`).
2. Öffnen Sie die Datei `constants/assets.ts`.
3. Ersetzen Sie die Unsplash-URL durch den lokalen Pfad, zum Beispiel:
   `slide1: "./assets/images/hero-alarm.jpg"`

Das System ist so vorbereitet, dass alle Komponenten automatisch das neue Bild übernehmen.
